메모리아 개인정보처리방침

시행일: 2026년 6월 9일

메모리아는 사용자가 암기 카드를 만들고 복습할 수 있도록 돕는 학습 앱입니다. 이 문서는 메모리아가 어떤 정보를 수집하거나 저장하고, 이를 어떻게 사용하며, 사용자가 데이터를 삭제할 수 있는 방법을 설명합니다.

앱 이름: 메모리아

운영자/개발자: youwon35

개인정보 문의 이메일: youwon35@gmail.com

앱 내 문의 경로: 앱 정보 > 문의하기

1. 수집 또는 저장되는 정보

  • 사용자가 직접 입력하거나 파일로 가져온 카드 앞면/뒷면, 폴더 정보
  • 학습 기록, 오답 기록, 일일 학습 목표, 카드 숨김 상태 등 학습 기능에 필요한 정보
  • Google 로그인 사용 시 계정 이메일과 사용자 식별자
  • 문의하기 사용 시 답변 받을 이메일, 문의 분류, 문의 내용, 접수 상태
  • 로그인 사용자의 앱 실행 기록 같은 운영 지표

2. 이용 목적

  • 카드 저장, 폴더 관리, 기기 간 동기화 제공
  • 학습 기록 확인, 오답 복습, 맞춤 문제 출제, 일일 목표 관리
  • 문의 응대와 앱 데이터 삭제 요청 처리
  • 앱 안정성 확인과 품질 개선

3. 제3자 처리

메모리아는 사용자의 카드 내용과 문의 내용을 판매하지 않습니다. 다만 로그인, 동기화, 데이터 저장, 문의 처리를 위해 Supabase, Google 등 앱 운영에 필요한 서비스 제공업체가 데이터를 처리할 수 있습니다.

4. 보안

동기화, 로그인, 문의 데이터는 HTTPS 등 암호화된 통신을 통해 전송됩니다. 사용자는 Google 계정과 기기 보안을 직접 관리해야 합니다.

5. 보관 및 삭제

로컬 데이터는 앱 삭제 또는 앱 데이터 삭제 시 기기에서 제거됩니다. Google 로그인으로 동기화된 카드, 폴더, 학습 기록, 오답 기록, 일일 학습 목표, 문의 기록, 계정 연결 운영 지표, 동기화 데이터는 앱 정보의 계정 / 데이터 삭제 기능에서 삭제할 수 있습니다. 이 작업은 메모리아에 저장된 앱 데이터만 삭제하며 Google 계정 자체는 삭제하지 않습니다.

앱을 사용할 수 없는 경우 계정 및 데이터 삭제 안내에 따라 이메일로 삭제를 요청할 수 있습니다. 법적 의무, 보안, 분쟁 대응을 위해 필요한 정보는 필요한 기간 동안 보관될 수 있습니다.

6. 아동의 개인정보

메모리아는 어린이를 주 대상으로 만들어진 앱이 아닙니다. 사용자가 미성년자인 경우 보호자의 지도 아래 앱을 사용하기를 권장합니다.

7. 변경

정책이 변경되면 앱 또는 이 공개 정책 페이지를 통해 안내합니다.
