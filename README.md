# MEMORIA policy pages

This repository hosts public policy pages for the MEMORIA Android app.

## Google Play Console URLs

Privacy Policy URL:

```text
https://youwon35.github.io/memoria-privacy/
```

Account and data deletion URL:

```text
https://youwon35.github.io/memoria-privacy/data-deletion.html
```

## Files

- `index.html`: 메모리아 개인정보처리방침
- `data-deletion.html`: 메모리아 계정 및 데이터 삭제 안내

## GitHub Pages setup

In GitHub repository settings, enable Pages with:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`
